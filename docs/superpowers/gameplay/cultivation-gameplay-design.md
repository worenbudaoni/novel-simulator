# 修仙模拟器玩法 — 系统映射方案

> 参考《了不起的修仙模拟器》玩法机制，映射到现有互动叙事系统
> 本方案不做代码改动，仅设计

---

## 一、世界观与玩家身份

玩家扮演一名**初入修仙界的散修**，在修真世界中探索、修炼、抉择，最终走向飞升或陨落。

---

## 二、属性系统映射

### 2.1 当前属性 → 修仙属性

| 当前属性 | 修仙映射 | 作用 |
|---------|---------|------|
| HP | 气血 | 归零则陨落（死亡） |
| 攻击 | 攻击力 | 战斗能力，功法+法宝加成 |
| 防御 | 防御力 | 抵御伤害，护体法宝加成 |
| 智力 | 悟性 | 修炼速度、领悟功法概率、解锁隐藏选项 |
| 魅力 | 道心 | 社交成功率、心魔抵抗、双修效果 |
| 运气 | 气运 | 奇遇概率、天劫难度、宝物获取 |

### 2.2 新增属性（未来扩展）

| 属性 | 说明 |
|------|------|
| 修为值 | 当前境界修炼进度，满值后可突破 |
| 境界 | 练气→筑基→金丹→元婴→化神→渡劫 |

未增加前，可以用 `choicesMade * 10` 模拟修为，`intelligence` 映射境界门槛。

## 三、节点与选项设计

### 3.1 节点类型

| 节点类型 | 说明 | 选项数量 |
|---------|------|---------|
| 修炼 | 安全修炼，增加修为 | 2-3 |
| 秘境 | 探索秘境，高回报高风险 | 3-4 |
| 城镇 | 社交、交易、接任务 | 3-4 |
| 宗门 | 加入/拜师/学艺 | 2-3 |
| 渡劫 | 境界突破，决定性节点 | 2-3（含结局） |
| 结局 | 飞升/陨落/散仙 | 0 |

### 3.2 选项示例

```
节点：「幽暗秘境」
├─ 谨慎探索 → 安全但有收获（常规）
├─ 深入腹地 → 高概率触发转盘（冒险）
├─ 破解阵法 → 需悟性≥60（属性解锁）
└─ 布置传送阵 → 保留退路（仅限金丹以上）
```

### 3.3 境界解锁路线

```
练气期 → 筑基期 → 金丹期 → 元婴期 → 化神期 → 渡劫飞升
  └→ 陨落     └→ 散仙    └→ 夺舍
```

- 每个大境界跨越需要经过 1-2 个关键节点
- 突破有成功率，与悟性、气运相关
- 突破失败可能触发转盘（心魔/天劫）

## 四、转盘扇区（修仙版）

| 扇区 | 概率 | 效果 |
|------|------|------|
| ✨ 奇遇 | 20% | 获得功法/法宝/丹药，属性全面提升 |
| 💎 天材地宝 | 20% | 获得修炼资源，修为大增 |
| ⚔️ 妖兽袭击 | 20% | 战斗，气血下降，但战后有收获 |
| 💀 心魔 | 15% | 道心受损，悟性暂时下降 |
| 🌀 机缘 | 15% | 悟性/气运永久提升 |
| 🌩️ 天劫 | 10% | 境界突破判定（高难度节点专属） |

## 五、LLM 故事风格（StoryChain）

### 5.1 当前风格

```
你来到了迷雾森林。四周的环境让你感到既熟悉又陌生。
```

### 5.2 修仙风格

```
你盘膝而坐，引导灵气沿着经脉缓缓运转。丹田内的真元如涓涓细流，
滋养着四肢百骸。忽然，你感应到百里外传来一阵异样的灵气波动——
那是天材地宝出世的气息，还是某位大能正在斗法？
```

### 5.3 参考变量

StoryChain 生成时可参考以下上下文：

```
- 当前境界（由 intelligence 映射）
- 当前修为（由 choicesMade 映射）
- 气血状态（hp）
- 当前事件（如有触发的转盘结果）
- 历史选择（historyPath）
```

## 六、修仙主题节点标题示例（12个节点）

```
1. 灵根觉醒（起点）
2. 初入修仙界
3. 散修坊市
4. 苍狼山脉（历练）
5. 上古遗迹（秘境）
6. 七煞门（宗门选择）
7. 丹霞洞天（修炼）
8. 妖兽围攻（战斗节点）
9. 拍卖大会（社交/交易）
10. 天劫降临（突破节点）
11. 飞升之路（结局分支）
12. 陨落之渊（结局）
```

## 七、与现有系统的兼容性

| 系统 | 改动 | 工作方式 |
|------|------|---------|
| Node 表 | 无需改动 | minIntelligence 直接映射悟性门槛 |
| NodeOption | 无需改动 | 3-4个选项，部分设属性门槛 |
| RandomEvent | 无需改动 | event_type 映射扇区类型 |
| CharacterPanel | CSS 显示优化 | 属性标签改为修仙命名 |
| StoryChain | prompt 调整 | 改为修仙风格叙事 |
| EventChain | prompt 调整 | 改为修仙主题事件 |
| 现有数据 | 无需迁移 | 新主题从新建作品开始 |
