# P3 核心玩法

> 对应总设计: `../specs/2026-06-18-novel-interactive-story-simulator-design.md`
> 预估工时: ~3 周
> 验收标准: 用户可以选择作品、探索节点地图、做出选择/转盘抽奖、LLM 生成故事、存档读档完整跑通

---

## 本阶段目标

完成 Player 端的核心游戏体验，包括作品选择、节点地图探索、选项面板、转盘抽奖、LLM 事件和故事生成、SSE 流式输出以及存档功能。

### 3.13 P3-C: LLM 深度参与

- [x] StoryChain LLM 调用
  - 全量 OpenAI chat 对话历史（Redis）
  - 每次发送完整 system + user + assistant 历史
  - 生成故事后追加到对话历史
  
- [x] EventChain LLM 调用
  - 读取共享对话历史
  - LLM 返回 JSON（title, content, 6维属性变化）
